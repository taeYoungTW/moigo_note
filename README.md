1. 문제의식

- Action 구성에 대한 문제
  : 스토어의 Action에서 연관된 기능까지 구현 하여 묶어야 하는가?
  아니면, 독립성을 위해서 사용되는 컴포넌트 내부에서 함수를 통해서 연관되 기능 끼리의 조합을 해야하는가?
- Global states, actions들이 컴포넌트의 Local states, functions의 식별자 중복 가능성이 있기 때문에 `_`를 붙여 구분하여 사용할지 생각
